Jpa/hibernate/spring Data/spring Data JpaйғҪжҳҜд»Җд№Ҳ

2017-07-22 spring 0 иҜ„и®ә еӯ—ж•°з»ҹи®Ў: 510(еӯ—) йҳ…иҜ»ж—¶й•ҝ: 1(еҲҶ)

д»Җд№ҲжҳҜJPA

  • JPAе…Ёз§°Java Persistence API.JPAйҖҡиҝҮJDK 5.0жіЁи§ЈжҲ–XMLжҸҸиҝ°еҜ№иұЎпјҚе…ізі»иЎЁзҡ„жҳ е°„е…ізі»пјҢе№¶е°ҶиҝҗиЎҢжңҹзҡ„е®һдҪ“еҜ№иұЎжҢҒд№…еҢ–еҲ°ж•°жҚ®еә“дёӯгҖӮ
  • JPA(Java Persistence API)жҳҜSunе®ҳж–№жҸҗеҮәзҡ„JavaжҢҒд№…еҢ–и§„иҢғгҖӮе®ғдёәJavaејҖеҸ‘дәәе‘ҳжҸҗдҫӣдәҶдёҖз§ҚеҜ№иұЎ/е…ізі»жҳ е°„е·Ҙе…·жқҘз®ЎзҗҶJavaеә”з”Ёдёӯзҡ„е…ізі»ж•°жҚ®гҖӮ

д»Җд№ҲжҳҜHibernate

  • HibernateжҳҜдёҖдёӘејҖж”ҫжәҗд»Јз Ғзҡ„еҜ№иұЎе…ізі»жҳ е°„жЎҶжһ¶пјҢе®ғеҜ№JDBCиҝӣиЎҢдәҶйқһеёёиҪ»йҮҸзә§зҡ„еҜ№иұЎе°ҒиЈ…пјҢе®ғе°ҶPOJOдёҺж•°жҚ®еә“иЎЁе»әз«Ӣжҳ е°„е…ізі»пјҢжҳҜдёҖдёӘе…ЁиҮӘеҠЁзҡ„ormжЎҶжһ¶пјҢhibernateеҸҜд»ҘиҮӘеҠЁз”ҹжҲҗSQLиҜӯеҸҘпјҢиҮӘеҠЁжү§иЎҢпјҢдҪҝеҫ—JavaзЁӢеәҸе‘ҳеҸҜд»ҘйҡҸеҝғжүҖж¬Ізҡ„дҪҝз”ЁеҜ№иұЎзј–зЁӢжҖқз»ҙжқҘж“Қзәөж•°жҚ®еә“

д»Җд№ҲжҳҜSpring Data

  • Spring DataжҳҜдёҖдёӘз”ЁдәҺз®ҖеҢ–ж•°жҚ®еә“и®ҝй—®пјҢе№¶ж”ҜжҢҒдә‘жңҚеҠЎзҡ„ејҖжәҗжЎҶжһ¶гҖӮе…¶дё»иҰҒзӣ®ж ҮжҳҜдҪҝеҫ—ж•°жҚ®еә“зҡ„и®ҝй—®еҸҳеҫ—ж–№дҫҝеҝ«жҚ·пјҢе№¶ж”ҜжҢҒmap-reduceжЎҶжһ¶е’Ңдә‘и®Ўз®—ж•°жҚ®жңҚеҠЎгҖӮжӯӨеӨ–пјҢе®ғиҝҳж”ҜжҢҒеҹәдәҺе…ізі»еһӢж•°жҚ®еә“зҡ„ж•°жҚ®жңҚеҠЎпјҢеҰӮOracle RACзӯүгҖӮеҜ№дәҺжӢҘжңүжө·йҮҸж•°жҚ®зҡ„йЎ№зӣ®пјҢеҸҜд»Ҙз”ЁSpring DataжқҘз®ҖеҢ–йЎ№зӣ®зҡ„ејҖеҸ‘пјҢе°ұеҰӮSpring FrameworkеҜ№JDBCгҖҒORMзҡ„ж”ҜжҢҒдёҖж ·пјҢSpring Dataдјҡи®©ж•°жҚ®зҡ„и®ҝй—®еҸҳеҫ—жӣҙеҠ ж–№дҫҝгҖӮ

д»Җд№ҲжҳҜSpring Data JPA

  • Spring Data JPAеҸҜд»ҘжһҒеӨ§зҡ„з®ҖеҢ–JPAзҡ„еҶҷжі•пјҢеҸҜд»ҘеңЁеҮ д№ҺдёҚз”ЁеҶҷе®һзҺ°зҡ„жғ…еҶөдёӢпјҢе®һзҺ°еҜ№ж•°жҚ®зҡ„и®ҝй—®е’Ңж“ҚдҪңгҖӮйҷӨдәҶCRUDеӨ–пјҢиҝҳеҢ…жӢ¬еҰӮеҲҶйЎөгҖҒжҺ’еәҸзӯүдёҖдәӣеёёз”Ёзҡ„еҠҹиғҪгҖӮ
  • Spring DataжҳҜдёҖдёӘејҖжәҗжЎҶжһ¶пјҢиҖҢеңЁиҝҷдёӘжЎҶжһ¶дёӯSpring Data JPAеҸӘжҳҜиҝҷдёӘжЎҶжһ¶дёӯзҡ„дёҖдёӘжЁЎеқ—пјҢжүҖд»ҘеҗҚз§°жүҚеҸ«Spring Data JPAгҖӮеҰӮжһңеҚ•зӢ¬дҪҝз”ЁJPAејҖеҸ‘пјҢдҪ дјҡеҸ‘зҺ°иҝҷдёӘд»Јз ҒйҮҸе’ҢдҪҝз”ЁJDBCејҖеҸ‘дёҖж ·жңүзӮ№зғҰдәәпјҢжүҖд»ҘSpring Data JPAзҡ„еҮәзҺ°е°ұжҳҜдёәдәҶз®ҖеҢ–JPAзҡ„еҶҷжі•пјҢи®©дҪ еҸӘйңҖиҰҒзј–еҶҷдёҖдёӘжҺҘеҸЈз»§жүҝдёҖдёӘзұ»е°ұиғҪе®һзҺ°CRUDж“ҚдҪңдәҶгҖӮ
